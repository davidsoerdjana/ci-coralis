Cara menjalankan:
Buka XAMPP, nyalakan Apache & MySQL
Masuk ke browser ketik localhost/ci-coralis

Step membuat project: referensi documentation https://codeigniter.com/userguide3/index.html
1. Buka application > config > autoload.php, buat *1 dan config.php buat lanjutan *1. Lalu buat file .htaccess didalam folder project,
   referensi https://codeigniter.com/userguide3/general/urls.html?highlight=htaccess
2. Buka application > config > routes.php, buat *2
3. Masuk ke folder controllers, didalamnya buat file Auth.php, masuk ke dalam file itu buat *3
4. Masuk ke folder vendor > sbadmin2. Copy folder css, img, js, vendor kedalam folder assets di dalam project
5. Masuk ke application > views, buat folder auth dan templates. Didalam folder auth buat file login.php,
   didalam folder templates buat file auth_header.php dan auth_footer.php. Masuk ke dalam file auth_header.php buat *4,
   lalu masuk ke login.php dan auth_footer.php buat lanjutan *4
6. Masuk ke application > views > auth. Didalam folder auth buat file registration.php, masuk ke dalam file itu buat *5
7. Buka browser, masuk ke localhost/phpmyadmin buat database dengan nama codeigniter_coralis atau ci_coralis,
   lalu masuk ke application > config > database.php isi 'username' dengan 'root' dan 'database' dengan 'nama_database_yang_dibuat_di_phpmyadmin_tadi'
8.
